# pac-man
Easy version of [Pac Man](https://en.wikipedia.org/wiki/Pac-Man). Eat dots and escape from Pinky! Implemented with Python 3 and [Processing 3](https://github.com/processing/processing/releases).
![pacman demo](https://github.com/user-attachments/assets/52551cb0-836b-41c7-9341-c9318797fd83)

# Play
1. Download [Proceesing](https://github.com/processing/processing/releases).
2. Open Processing and click "Add Mode" and install Python Mode.
<img width="1120" height="735" alt="image" src="https://github.com/user-attachments/assets/ccd0e381-69bf-4d60-ad75-482a96807fbf" />
<img width="922" height="740" alt="image" src="https://github.com/user-attachments/assets/48d7ac93-abad-4767-a1a5-c490c3ad4901" />

3. Open "pacman_game.pyde" with Processing and add other files in the sketch.
4. Click the Run button and enjoy!
